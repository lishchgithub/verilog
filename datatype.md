- 常量
	- 数字-整数：<位宽><进制><数字>
		- 位宽缺省为系统默认位宽（32）
		- 进制缺省为10进制（d）
		- Ex.：8'b01010101,4'd19,2'h1a
		- x表示不定值，z或者？表示高阻值（对下级电路没有任何输入）
		- 负数，只能放在表达式前面
		- 下划线提高阅读效率无实际意义，16'b0001_1000_0100_0010
	- 参数parameter：
		- parameter x=1；
		- parameter z=x+y;
		- 可通过参数传递

+ 变量
	+ wire型，信号变量
	+ reg型，寄存器变量
	+ memory型，存储器建模，由reg扩展而成
		+ reg[7:0] mema[255:0]
